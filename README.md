# JR北海道風 発車標 PIDS

Minecraft 1.20.1 / MTR 4 / Joban Client Mod v2 の Scripted PIDS Preset です。

## 使い方

1. このフォルダをリソースパックとして有効化します。
2. `F3 + T` でリソースを再読み込みします。
3. JCMのPIDSをMTRブラシで右クリックします。
4. 表示形式から使用するプリセットを選択します。
   - 通常の発車標: 「JR北海道風 発車標」
   - 鉄路願景LCD発車標: 「JR北海道風 発車標（鉄路願景LCD）」
5. PIDSが検出するホームを設定します。

2つのプリセットはブロック種類に関係なく選択画面へ個別に表示され、選択したプリセットによってレイアウトが切り替わります。

先発・次発列車の種別（系統番号）、発車時刻、行先、番線をMTRの到着情報から表示します。

PIDS設定画面の「発車標に表示したいメッセージ」1段目が、発車標上部の方面案内に表示されます。

2段目のメッセージは2つ目の列車行に表示されます。

- 2段目の「行き先などを非表示にする」が有効: メッセージを常時表示
- 無効: 2つ目の列車とメッセージを6秒ごとに交互表示
- 長いメッセージ: 左方向へスクロール
- 「番線を非表示」が有効: 番線の数字を非表示

先発列車の到着25秒前になると、2つ目の列車行に「列車がまいります。」を点滅表示します。この表示は、列車表示と2段目メッセージより優先されます。

ホーム向け通常版では、メッセージ入力1段目の「行き先などを非表示」が有効な場合、到着警告を表示しません。

表示する列車がない場合、1つ目の列車行に「調整中」と表示します。

鉄路願景LCD発車標では専用の4段表示になります。

- 上部: 1段目メッセージ（未入力時は「発車ご案内 Train Infomation」）
- 1段目: 先発列車
- 2段目: 先発列車の停車駅
- 3段目: 次発列車
- 4段目: 次々発列車／メッセージ入力2段目

先発列車の停車駅が長い場合は左方向へスクロールします。

LCD版の4段目は、メッセージ入力2段目の「行き先などを非表示」が有効ならメッセージを固定表示し、無効なら次々発列車と6秒ごとに交互表示します。

## 固定文の変更

`assets/jsblock/joban_custom_resources.json` の `scriptInput` を編集してください。

- `directionText`: PIDS側でメッセージが未入力の場合の上段案内（既定値: `発車ご案内 Train Infomation`）
- `noTrainText`: 列車がない場合の表示
- `backgroundColor`: 背景色の16進RGB値
- `arrivalWarningSeconds`: 到着警告を開始する到着前秒数
- `arrivalWarningBlinkIntervalMs`: 到着警告の点滅間隔（ミリ秒）
- `arrivalWarningText`: 到着警告の文章

編集後は `F3 + T` で再読み込みしてください。
